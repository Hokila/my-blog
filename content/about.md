## 歡迎來到我的部落格

您好！我是 Hokila，這是我的個人技術部落格。

## 關於這個 Blog

這個部落格是我分享技術心得、學習筆記和生活感想的地方。我使用 Hugo 靜態網站生成器搭配 Mainroad 主題建立，並整合了 Giscus 評論系統，讓讀者可以透過 GitHub 帳號留言互動。

## 技術棧

- **靜態網站生成器**: Hugo
- **主題**: Mainroad
- **評論系統**: Giscus (基於 GitHub Discussions)
- **部署平台**: GitHub Pages
- **版本控制**: Git & GitHub

## 興趣領域
